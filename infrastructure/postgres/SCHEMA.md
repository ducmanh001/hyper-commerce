## Business Rules (not in code)

### Monetary values

- All VND prices stored as `BIGINT` (integer dong) — NEVER float/decimal
- Balance derived: `SELECT balance_after FROM wallet_transactions ORDER BY created_at DESC LIMIT 1` — never mutable column

### Concurrency rules

- Inventory: stock decrement via Lua script only (atomic) — never two Redis calls
- Wallet: `SELECT ... FOR UPDATE` on latest tx row before every credit/debit

### Loyalty tiers

- BRONZE 0–4.9M VND → 1% cashback
- SILVER 5M–19.9M → 1.5% | GOLD 20M–99.9M → 2% | PLATINUM 100M+ → 3%

### Live gifts (coin split)

- ROSE=10c | HEART=50c | DIAMOND=500c | CROWN=1000c | ROCKET=5000c
- Host receives 70%, platform 30%

### Feed ranking weights

- variant-1: relevance=0.5 · recency=0.3 · reputation=0.2
- variant-2: relevance=0.6 · recency=0.2 · reputation=0.2
- Celebrity pull threshold: follower_count > 10,000

### Redis key patterns (service-wide)

```
inv:stock:{productId}:{variantId}           TTL=15min (reservation)
flash:stock:{saleItemId}                    TTL=sale_duration
flash:user:bought:{saleItemId}:{userId}     TTL=24h
feed:feat:user:{userId}                     TTL=300s
feed:feat:seller:{sellerId}                 TTL=600s
feed:seen:{userId}                          TTL=86400s
feed:ab:{userId}                            TTL=604800s
referral:code:{code}                        TTL=365d
wallet:rl:topup:{userId}                    TTL=3600s (rate limit)
```
